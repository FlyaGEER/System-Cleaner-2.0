============================================
     SYSTEM CLEANER PRO v3.0 - ИНСТРУКЦИЯ
============================================

ДЛЯ ЗАПУСКА ПРОГРАММЫ:

1. Запустите "Запуск.bat" или "SystemCleaner.bat"
2. Если Windows спрашивает разрешение - нажмите "Да"
3. Если требуется запуск от администратора - согласитесь

ГЛАВНОЕ МЕНЮ ПРОГРАММЫ:

[1] БЫСТРАЯ ОЧИСТКА
    Автоматически очищает:
    • Временные файлы Windows
    • Кэш браузеров (Chrome, Edge, Firefox)
    • Корзину на всех дисках
    • DNS и системные кэши

[2] РАСШИРЕННАЯ ОЧИСТКА
    Выбор компонентов для очистки

[3] ОЧИСТКА БРАУЗЕРОВ
    Только кэш браузеров

[4] ОЧИСТКА СИСТЕМЫ
    Системные файлы и кэши

[5] ДОПОЛНИТЕЛЬНЫЕ ИНСТРУМЕНТЫ
    Дефрагментация, проверка диска и др.

[6] НАСТРОЙКИ
    Настройки программы

[7] СПРАВКА
    Информация и помощь

[0] ВЫХОД
    Завершение программы

ВАЖНО:
• Перед очисткой закройте все браузеры
• Программа безопасна для системы
• Для возврата в главное меню нажмите 0
• Для выхода из программы нажмите 0
