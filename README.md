# System Cleaner Pro 3.0

Профессиональный инструмент для очистки Windows. Бесплатная альтернатива CCleaner с открытым исходным кодом.

## 📋 Функции программы

### 🚀 Быстрая очистка
- Очистка временных файлов (`%TEMP%`, `%TMP%`)
- Очистка корзины на всех дисках
- Удаление временных файлов Windows (`C:\Windows\Temp`)
- Очистка Prefetch

### 🔧 Расширенная очистка
- Глубокая очистка системного кэша
- Очистка DNS кэша
- Очистка кэша эскизов (thumbnails)
- Очистка кэша Windows Update
- Удаление старых точек восстановления (опционально)

### 🌐 Очистка браузеров
- **Google Chrome** (кэш, куки, история, временные файлы)
- **Microsoft Edge** (кэш, куки, история)
- **Mozilla Firefox** (кэш, куки, история)
- **Internet Explorer** (временные файлы)

### 📦 Дополнительно
- Очистка логов Windows
- Очистка кэша Office
- Очистка недавних документов
- Очистка буфера обмена
- Очистка кэша DirectX
- Очистка миниатюр

## ⚙️ Как использовать

### Вариант 1: Установочная версия (рекомендуется)
1. Скачай `SystemCleaner_Setup.exe`
2. Запусти установщик от имени администратора
3. Следуй инструкциям мастера установки
4. Используй ярлык на рабочем столе

### Вариант 2: Портативная версия
1. Скачай `SystemCleaner.bat`
2. Сохрани файл на компьютер
3. Нажми правой кнопкой → "Запуск от имени администратора"

## 📌 Системные требования
- **ОС:** Windows 7/8/10/11 (32/64-bit)
- **Права:** Администратора
- **Место:** 50 МБ свободного места
- **Зависимости:** .NET Framework 4.5+ (для установщика)

## ⚠️ Важно
- Всегда запускай программу **от имени администратора**
- Закрой браузеры перед очисткой
- Рекомендуется создать точку восстановления перед первой очисткой
- Программа **не удаляет** личные файлы (документы, фото, видео)
