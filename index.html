ЧАСТЫЕ ВОПРОСЫ

Антивирус ругается на BAT-файл?
Это ложное срабатывание. BAT-файлы часто помечаются антивирусами, потому что работают с системными папками. Добавьте файл в исключения или используйте установочную версию .exe. Программа абсолютно безопасна, весь код открыт на GitHub.
Можно восстановить удаленные файлы?
Программа удаляет только временные файлы и кэш. Личные документы, фото и важные данные не затрагиваются. Если вы случайно удалили что-то важное — попробуйте специальные программы для восстановления (например, Recuva).
Чем отличается .exe от .bat?
.exe — установочная версия с мастером установки и ярлыками. Рекомендуется для постоянного использования.
.bat — портативный скрипт, работает без установки, можно носить на флешке.
Программа бесплатная?
Да, System Cleaner Pro полностью бесплатен. Никаких скрытых платежей, подписок или рекламы. Open source проект для сообщества.
Какие файлы удаляются?
Удаляется: временные файлы Windows, кэш браузеров, DNS кэш, эскизы, логи системы, корзина.
Не удаляется: документы, фото, видео, музыка, рабочий стол.
Почему нужны права администратора?
Для доступа к системным папкам (Windows\Temp, Prefetch, DNS кэш) требуются права администратора. Запускайте файл правой кнопкой → "Запуск от имени администратора".
Какие Windows поддерживаются?
Windows 7, 8, 8.1, 10, 11 (32-bit и 64-bit). Требуются права администратора.
Как часто запускать очистку?
Рекомендуется раз в 1-2 месяца. При активном использовании компьютера можно чаще.
