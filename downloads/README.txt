📥 СПОСОБЫ ПОЛУЧЕНИЯ ПРОГРАММЫ:

1. УСТАНОВЩИК (Setup.exe) - РЕКОМЕНДУЕМ:
   • Скачайте Setup.exe с сайта
   • Запустите файл
   • Нажмите "Да" при запросе прав администратора
   • Следуйте инструкциям установщика
   • После установки ярлык появится на рабочем столе

2. ПОРТАТИВНАЯ ВЕРСИЯ (SystemCleaner.bat):
   • Скачайте SystemCleaner.bat
   • Нажмите ПРАВОЙ кнопкой мыши на файле
   • Выберите "Запуск от имени администратора"
   • Для удобства создайте ярлык на рабочем столе

🛠️ КАК УДАЛИТЬ ПРОГРАММУ:

Если устанавливали через Setup.exe:
1. Откройте Панель управления
2. Перейдите в "Программы и компоненты"
3. Найдите "System Cleaner Pro"
4. Нажмите "Удалить"

Если использовали портативную версию:
1. Просто удалите файлы программы
